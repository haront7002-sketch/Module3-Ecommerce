# Module3-Ecommerce

Full-stack event booking app with:
- `Frontend/`: Vue 3 + Vite + Vuex
- `Backend/`: Express + MySQL

## Features
- Auth (register/login/JWT)
- Explore events (search/filter/date + load all)
- Map events (distance radius + load all)
- Favourites and cart
- Checkout/payments and ticket generation
- Profile editing and preferences

## Quick Start

1. Start backend:
```sh
cd Backend
npm install
npm run dev
```

2. Start frontend (new terminal):
```sh
cd Frontend
npm install
npm run dev
```

3. Open app:
- `http://localhost:5173`
