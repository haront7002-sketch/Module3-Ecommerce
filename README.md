# Module3-Ecommerce

Full-stack ecommerce project with a Vue 3 frontend and an Express/MySQL backend.

## Project Structure

- `Frontend/` - Vue 3 + Vite client app
- `Backend/` - Express API + MySQL

## Quick Start

### 1) Backend

```sh
cd Backend
npm install
npm run dev
```

### 2) Frontend

Open a second terminal:

```sh
cd Frontend
npm install
npm run dev
```
